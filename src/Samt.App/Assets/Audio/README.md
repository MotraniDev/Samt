# Adhan audio assets

Place a licensed adhan recording here as one of:

- `adhan.mp3`
- `adhan.wav`
- `adhan.m4a`

Set `DefaultAudio.Source` to `Bundled` in settings (or use `LocalFile` with an absolute path).

Until a licensed file is present, SAMT plays a short soft alert tone (`WindowsDefault`) generated under `%LocalAppData%\SAMT\tones\`.

**Do not** ship unlicensed commercial adhan recordings.
