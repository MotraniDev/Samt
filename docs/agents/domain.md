# Domain Docs

Single-context repo.

## Before exploring, read

- `CONTEXT.md` at the repo root
- `docs/adr/` when present
- `AGENTS.md` for Latin digits and skill shortlist

If files are missing, proceed silently.

## Vocabulary

Use terms from `CONTEXT.md`. Prefer project names over synonyms:

| Use | Avoid |
|-----|--------|
| PrayerSchedule | timetable blob |
| LocationProfile | city record |
| CalculationProfile | method settings bag |
| NotificationRule | alert config |
| LatinDigits | "English numbers" alone |
